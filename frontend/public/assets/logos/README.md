# SNSU Logo Directory

## Logo Files Required

Place your SNSU (Samar State University) official logos in this directory:

### Required Formats:
1. **snsu-logo.png** - Main logo (square, 512x512px recommended)
2. **snsu-logo-white.png** - White version for dark backgrounds
3. **snsu-logo-horizontal.png** - Horizontal layout for headers
4. **snsu-banner.png** - Banner version (1200x630px for social sharing)

### Optional Formats:
- **snsu-logo.svg** - Vector version (scalable, preferred)
- **snsu-favicon.ico** - Favicon for browser tabs
- **snsu-logo-small.png** - Thumbnail version (128x128px)

## Usage in App

```tsx
// Import in React components:
import logo from '/assets/logos/snsu-logo.png';

// Or use direct path:
<img src="/assets/logos/snsu-logo.png" alt="SNSU Logo" />
```

## Logo Guidelines

- Use official SNSU branding colors
- Maintain aspect ratio when scaling
- Include transparent background for PNG files
- Optimize file size for web (use tools like TinyPNG)
